# MITTRE-Attack-Dashboard

Dies ist ein Dashboard zur Visualisierung von Sicherheitsereignissen basierend auf MITRE ATT&CK Techniken.

## Screenshot

![Dashboard Screenshot](mittre-attack-dashboard.png)

## Features
- High / Medium / Low Threat Übersicht
- Detaillierte Event-Tabelle mit MITRE Technik-ID
- Modernes Dashboard Design mit Django
